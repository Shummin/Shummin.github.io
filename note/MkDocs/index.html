GitHub Pagas¶

GitHubによる，静的サイトのホスティングサービス． GitHubのアカウントがあれば，無料で簡単なウェブページを置くことができます． Webページを置く場所がないという人 or 諸々の理由で大学などに置きたくないという人にはおすすめです． 所属機関が管理するサイトにWebページを置いた場合，異動したときに移行しなくて済むので，個人的にはおすすめです．
